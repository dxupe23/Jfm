How to Add Your Images

1

Create Your Folders

Create a folder for your website. Inside it, create a subfolder named images.

portfolio-website/
├── index.html
└── images/
    ├── logo.png
    ├── profile.jpg
    └── project1.jpg
2

Save Your Images

Save your images with these exact filenames in the images folder:

  • Logo: logo.png
  • Profile picture: profile.jpg
  • Project image: project1.jpg
3

Replace Placeholders

The website will automatically display your images once they're in the correct folder with the correct names. No code changes needed!
